## 이메일 7: 분기 서신 발행 통지

**제목:**
```
펀드 업데이트 {Q}분기 {YYYY}년 — The Aurum Century Club
```

**프리헤더:**
```
{YYYY}년 {Q}분기 펀드 업데이트가 회원 포털에 게재되었습니다.
```

**참고:** 해당 이메일의 아이브로우 레이블(단색, 금색)은 이미 `FUND UPDATE · 펀드 업데이트`로 구현되어 있습니다.

{YYYY}년 독립 보관소 검증 보고서가 회원 포털에 게재되었습니다.

**참고:** 해당 이메일의 아이브로우 레이블은 이미 `VAULT VERIFICATION · 보관 검증`으로 구현되어 있습니다.

## 등록 양식 — 제목 표기 원칙

- 귀하의 (honorific possessive): "귀하의 [X]" 형식을 기본으로 사용
- 느낌표 사용 금지
- 이모지 사용 금지
- 시스템 자동 발송임을 암시하는 표현 사용 금지 ("자동 발송된 이메일입니다" 등)
- 이메일 유형을 명확히 표현하되 광고성 문구 사용 금지

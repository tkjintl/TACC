## 2. Register Guide

모든 회원 대면(member-facing) 텍스트는 **합쇼체** (formal polite speech, 하십시오체) 기준으로 작성합니다. 구체적으로:

동사 어미는 "-합니다", "-입니다", "-하십시오", "-하시기 바랍니다"를 사용합니다. 반말(-해, -이야), 비격식체(-해요, -이에요), 스타트업·IT 업계에서 관행적으로 쓰이는 친근한 표현은 일절 사용하지 않습니다. 독자를 지칭할 때는 "귀하" 또는 "회원"을 사용하고 "당신", "너", "고객님"은 사용하지 않습니다. 의무 사항을 표현할 때는 "-하여야 합니다", 금지 사항은 "-하여서는 아니 됩니다"를 사용합니다. 이는 한국 법률 문서 및 사모 금융 문서의 표준 어법입니다.

## 3. Terms to Avoid

다음 표현은 한국어 카피에서 사용하지 않습니다:

| 사용 금지 표현 | 이유 |
|---|---|
| 혁신적인 (innovative) | 실체 없는 홍보 문구 |
| 편리한 / 간편한 (seamless, easy) | 소비자 금융 앱 언어 — 사모 펀드 클럽에 부적합 |
| 투명한 (transparent) — 단독 사용 시 | 구체적 내용 없는 홍보 주장; 구체적 사실과 결합해서만 허용 |
| 안전한 (safe) — 담보 수준 암시 시 | 원금 보전을 암시하는 표현 금지 |
| 높은 수익률 / 고수익 | PPM에 근거하지 않은 수익 주장 금지 |
| 최고의 / 최상의 (best-in-class, world-class) | 근거 없는 최상급 표현 금지 |
| 독보적인 / 유일무이한 | 근거 없는 최상급 표현 금지 |
| 솔루션 (solution) | IT·컨설팅 업계 용어 — 금융 문서에서 회피 |
| 플랫폼 (platform) — 제품 홍보 맥락에서 | 기술 스타트업 언어 |
| 글로벌 (global) — 단독 사용 시 | 구체성 없는 수식어 |
| 리얼타임 (real-time) — 단독 사용 시 | "현재 평가금액", "실시간" 등 구체적 표현으로 대체 |
| 온보딩 (onboarding) — 회원 대면 문구에서 | "가입 절차" 또는 "가입 진행" 사용 |
| 유저 (user) | "회원" 사용 |
| 콘텐츠 (content) | 구체적 명칭(서신, 보고서, 문서 등) 사용 |

## 4. Platform-Specific Rules

**날짜 형식**
한국어 컨텍스트에서는 YYYY년 MM월 DD일 형식을 사용합니다. 예: 2026년 9월 30일. 이메일 제목 및 UI 레이블에서는 YYYY년 Q분기 또는 YYYY년 (연도만) 형식을 사용합니다.

**숫자 형식**
USD 금액은 아라비아 숫자 + "USD" 접두사를 사용합니다. 예: USD 151,800. 한국원(KRW) 금액은 한국식 단위(억/만)를 사용합니다. 예: 약 2억 원. 수수료율 등 백분율은 아라비아 숫자 + % 기호를 사용합니다.

**회원 번호**
항상 `#XXX` 형식으로 표기합니다 (예: #042). 번역하지 않습니다.

**고유 명칭 — 번역 금지**
"Aurum", "Century Club", "Aurum Century Club", "The Kilo", "TACC", "TACC Pte Ltd", "Prism Bridge", "Malca-Amit", "LBMA", "KYC", "NDA", "VCC", "NAV", "LTV", "PPM", "LP"는 번역하지 않습니다. 한국어 설명이 필요한 경우 괄호 병기를 사용합니다.

**플랫폼 URL**
`www.theaurumcc.com`은 번역하지 않습니다.

**조항 번호 참조**
PPM §7.4, PPM §10.4 등 조항 번호는 그대로 유지합니다. "§" 기호를 "제7.4조"로 대체해도 무방하나 일관성을 유지해야 합니다. 현재 카피에서는 "PPM §7.4" 형식을 유지했습니다.

**플레이스홀더 토큰**
[MGMT_FEE_PCT], [CUSTODY_FEE_PCT], [SUBSCRIPTION_FEE_PCT], {Q}, {YYYY}, {{member_name}}, {PARTNER_NAME}, {{last_known_price}}, {{last_price_date}} 등 모든 플레이스홀더 토큰은 영문 원본 그대로 유지합니다. 번역하지 않습니다.

## 5. Flagged Passages — Native Speaker Review Recommended

다음 구절은 법적 정확성 또는 자연스러운 한국어 표현의 관점에서 원어민 또는 한국 법률·금융 전문가의 재검토를 권장합니다.

**[FLAG-1] main-pitch-ko.md — 섹션 3, LTV 구조**
"펀드 계약서는 최소 자기자본 비율 50%를 의무 조항으로 규정하고 있으며, 이는 대출 잔액이 LBMA 현물가 기준 금 포지션 가치의 50%를 초과할 수 없음을 의미합니다."
검토 사항: "최소 자기자본 비율"이 "equity covenant"의 가장 정확한 한국 금융 법률 용어인지 확인 필요. "최소 담보유지비율" 또는 "최소 자기자본 비율 유지 의무 조항"도 검토 바람.

**[FLAG-2] main-pitch-ko.md — 섹션 6, 회원 의무**
"LP 지분을 양도하거나 이전하려는 시도"에서 "이전"과 "양도"가 중복될 수 있음. 영문 원문 "transfer or assign"의 법적 구별(양도 vs. 이전)이 한국 법률 문맥에서 동일하게 적용되는지 한국 법률 전문가 확인 필요.

**[FLAG-3] faq-ko.md — Q3, 유동성 불가**
"기간 종료(연장 포함 최대 7년) 시점까지 보유 가능한 자금으로만 청약하시기 바랍니다"는 강한 주의문입니다. 한국 규제상 투자 권유 적합성 요건(적합성 원칙) 문구와 충돌하지 않는지 법률 검토 권장.

**[FLAG-4] faq-ko.md — Q4, 세무**
"연간 해외 계좌 신고"는 한국 국세청의 "해외금융계좌 신고제도"를 지칭하는 것으로 추정됩니다. 정확한 법령 명칭을 원어민 세무사가 확인해야 합니다. 현재 표현은 의도적으로 일반적 표현을 사용했습니다.

**[FLAG-5] faq-ko.md — Q10, 규제**
"별도의 한국 금융 규제를 적용받는 한국 거주 투자자"는 영문 원문의 "separate Korean financial regulation"을 옮긴 것입니다. 구체적으로 어떤 법률(자본시장법, 외국환거래법 등)인지는 의도적으로 명시하지 않았습니다. 법률 검토자가 범위를 확인하고 필요 시 구체화하십시오.

**[FLAG-6] empty-states-ko.md — 오류: 포트폴리오 — 금 가격 피드**
"마지막으로 확인된 LBMA 고시가"에서 "고시가"가 영문 "LBMA fix"(오전/오후 고시)에 상응하는 올바른 용어인지 확인 바람. "LBMA 공식 고시가" 또는 "LBMA 기준가"도 검토 가능.

**[FLAG-7] ui-strings.json — "Read" → "열람"**
영문 "Read"는 문서 읽기 행동을 뜻하는 UI 버튼 레이블입니다. "열람"(formal, document review)을 선택했으나 UI 컨텍스트에 따라 "읽기"(더 짧고 직접적) 또는 "확인"이 더 자연스러울 수 있습니다. 실제 버튼 위치를 확인 후 결정 바람.

**[FLAG-8] 전반 — "딜러 스프레드 (dealer spread)"**
"딜러 스프레드"는 로마자+한국어 혼용 표현입니다. "거래 수수료 차이" 또는 "매매 스프레드"도 검토 가능. 현재는 한국 금융계에서 관용적으로 쓰이는 "딜러 스프레드"를 채택했습니다.

**[FLAG-9] 전반 — "디레버리지 이벤트"**
"강제 디레버리지 이벤트" (forced deleveraging event)는 로마자 혼용 표현입니다. "강제 레버리지 해소 상황" 또는 "강제 담보 처분 상황"이 더 격식적일 수 있습니다. 본 문서에서는 "강제 디레버리지 이벤트"를 사용했으나 최종 검토자가 판단 바람.

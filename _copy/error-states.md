# Error States — Platform Copy
# Format: PAGE / COMPONENT: [eyebrow] / [headline] / [subtext]
# [NEEDS KO TRANSLATION] for deployed platform

---

## Documents — Download failed

**EYEBROW:** Download Error

**HEADLINE:** Document could not be retrieved.

**SUBTEXT:** The file is temporarily unavailable. Refresh the page and try again. If the problem persists after two attempts, contact your Administrator — the document remains on file and will be resent manually.

---

## Portfolio — Gold price feed unavailable

**EYEBROW:** Price Data

**HEADLINE:** Live price unavailable.

**SUBTEXT:** The gold price feed is currently unreachable. Portfolio values shown reflect the last confirmed LBMA fix: {{last_known_price}} as at {{last_price_date}}. Your position and bar registry are unaffected. The feed will update automatically when restored.

---

## Subscription Form — Signature name does not match legal name

**EYEBROW:** Signature Mismatch

**HEADLINE:** Name does not match.

**SUBTEXT:** The name entered in the signature field does not match the legal name on your subscription record. Please re-enter your full legal name exactly as it appears on your passport. This is required for KYC compliance and the subscription agreement to be valid.

---

## Login — Incorrect code

**EYEBROW:** Access Code

**HEADLINE:** Code not recognised.

**SUBTEXT:** The code you entered is incorrect. Check that you are using the most recent code sent to your registered email address. Codes are single-use; if you have already used this code, request a new one.

---

## Login — Code expired

**EYEBROW:** Access Code

**HEADLINE:** This code has expired.

**SUBTEXT:** Access codes expire after 15 minutes. Return to the sign-in page to request a new code. If you did not request this code, no action is needed — your account is unchanged.

---

## NDA Upload — File too large

**EYEBROW:** Upload Error

**HEADLINE:** File exceeds the size limit.

**SUBTEXT:** The maximum permitted file size for NDA upload is 20 MB. Please reduce the file size — scan at a lower resolution, or compress the PDF — and try again. The signed NDA must be legible; do not reduce quality below a readable threshold.

---

## NDA Upload — File type not accepted

**EYEBROW:** Upload Error

**HEADLINE:** File type not accepted.

**SUBTEXT:** Only PDF files are accepted for NDA upload. Convert your document to PDF and upload again. If you are uploading a scan, save it as a single-page or multi-page PDF rather than as an image file.

---

## Subscription — NDA must be approved before subscribing

**EYEBROW:** Action Required

**HEADLINE:** NDA approval is pending.

**SUBTEXT:** Your NDA has been received and is under review. Subscription documents are issued once the Manager has approved your NDA. This typically takes one to two business days. You will be notified by email when your NDA is approved and subscription is available.

---

## Session Expired — Please log in again

**EYEBROW:** Session

**HEADLINE:** Your session has ended.

**SUBTEXT:** For security, member sessions expire after a period of inactivity. Please sign in again to continue. No data has been lost.

---

## General Server Error

**EYEBROW:** System Error

**HEADLINE:** Something went wrong.

**SUBTEXT:** An unexpected error occurred. Refresh the page and try again. If the issue persists, contact your Administrator directly — your account and documents are not affected by platform errors.
